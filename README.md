## Kurulum
1. config.example.ini dosyasını config.ini olarak güncelleyin
2.  OpenWeatherMap (https://openweathermap.org/api)  API Sayfası üzerinden kayıt olup anahtar oluşturun     
3. config.ini dosyasını düzenleyin: 
 
   [API]
   api_key = OPENWEATHERMAP_ANAHTARINIZI_BURAYA_YAZIN  # Örnek: 5a4b3c2d1e0f9g8h7i6j5k4l3m2n1o0p

4. pip install -r requirements.txt ile gerekli kütüphaneleri kurulumu

5. python main.py  ile programı çalıştır

6. MÜHENDİSLİKTE BİLGİSAYAR UYGULAMALARI I DERSİ İÇİN YAPILMIŞTIR.
